========================================
GitHub Setup Guide for Emergency SOS App
========================================

STEP 1: Install Git
-------------------
1. Download Git from: https://git-scm.com/download/win
2. Run the installer
3. Use default settings
4. Verify installation: git --version


STEP 2: Initialize Git Repository
----------------------------------
Open PowerShell in the project folder and run:

git init
git add .
git commit -m "Initial commit: Emergency SOS App"


STEP 3: Create GitHub Repository
---------------------------------
1. Go to: https://github.com/new
2. Repository name: emergency-sos-app
3. Description: Emergency alert app with SMS, WhatsApp, and live tracking
4. Choose: Public or Private
5. DO NOT initialize with README (we already have one)
6. Click "Create repository"


STEP 4: Connect to GitHub
--------------------------
Replace YOUR_USERNAME with your GitHub username:

git remote add origin https://github.com/YOUR_USERNAME/emergency-sos-app.git
git branch -M main
git push -u origin main


STEP 5: Future Updates
-----------------------
After making changes:

git add .
git commit -m "Description of changes"
git push
